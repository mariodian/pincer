# v0.4.0

This release adds a force sync option for daemon-connected setups, giving you a clean slate when incremental sync isn't enough.

## Highlights

- **Force sync from daemon** — Discard all local data and pull a fresh copy from your daemon. Useful when local data gets out of sync or you want a clean start. A confirmation dialog and progress notifications keep you informed along the way. Close the window while it runs — you'll get a system notification when it's done.
- **Real-time sync status** — The settings page now shows live sync state so you always know what's happening.

## Fixes

- Fixed database sequence conflicts that could occur after a force sync cleared local data.
- Improved the force sync warning message for clarity.
