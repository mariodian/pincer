# v0.3.4-dev

> ⚠️ Development preview — not recommended for production use.

Introduces the background daemon for macOS monitoring continuity, and adds Linux daemon packaging to GitHub releases.

## Highlights

- **Background daemon** — Pincer can now run a background process (`pincerd`) that keeps monitoring active while the desktop app is closed, sleeping, or restarting
- **Linux daemon release** — each release now publishes a `pincerd-vX.Y.Z-linux-x64.tar.gz` bundle for Linux users
- **Daemon settings** — the app now shows connection testing and sync status feedback in settings when the daemon is enabled
