# v0.2.0

Adds Linux tray support and expands the agent types Pincer can monitor.

## Highlights

- **Linux tray icon** — Pincer's tray icon now works on Linux
- **New agent types** — added support for Hermes and OpenCode agent types

## Fixes

- Fixed the agent URL input field triggering unwanted browser autocomplete and spellcheck
- Fixed default port configuration for OpenClaw agents
- Fixed Linux tray behavior to use the native tray implementation correctly
