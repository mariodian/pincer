# v0.1.2

Keyboard accessibility improvements, app state persistence, and a Windows icon.

## Highlights

- **Keyboard shortcuts** — press Escape to close discard dialogs and navigate back in the agent form
- **Window state persistence** — Pincer now remembers your window size and position between sessions
- **Windows icon** — added a proper `.ico` icon for Windows
- Replaced icon library with Lucide for more consistent iconography across the app

## Fixes

- Fixed chart rendering edge cases for more reliable display
- Fixed disabled agents respecting your local visibility preference
