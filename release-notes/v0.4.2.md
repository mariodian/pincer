# v0.4.2

This release improves the macOS Homebrew installation experience for the Pincer daemon.

## Highlights

- **Smoother Homebrew setup** — The Homebrew formula now uses proper version interpolation and installs files to the correct locations. Setup instructions for configuring your daemon secret have been clarified with step-by-step guidance.

## Fixes

- Fixed an issue where the daemon could not find its migration files when installed via Homebrew, by adding support for the Homebrew configuration directory.
