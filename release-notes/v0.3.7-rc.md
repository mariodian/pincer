# v0.3.7-rc

> ⚠️ Release candidate preview — validate this build before production rollout.

This release candidate fixes a database migration issue that could leave some upgraded installs without the uniqueness safeguards needed for checks and incident events.

## Highlights

- Upgraded installs now repair missing database indexes automatically during startup
- Fresh installs apply the affected migrations correctly, reducing the risk of duplicate monitoring records

## Fixes

- Fixed a migration packaging issue that could prevent uniqueness rules from being created for status checks and incident events
