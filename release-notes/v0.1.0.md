# v0.1.0

First fully functional release of Pincer. Agents can now be created, edited, and monitored from the desktop app.

## Highlights

- **Agent editing** — agents can now be saved and updated from the agent form
- **Improved navigation** — app startup and page routing are more reliable
- **Tray integration** — clicking an agent in the tray now opens the main window
- **Broader logging** — improved error visibility across the app for easier debugging
