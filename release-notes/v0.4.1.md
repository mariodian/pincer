# v0.4.1

This release brings the Pincer daemon to macOS with Homebrew support, along with UI polish and a richer timeline view.

## Highlights

- **macOS daemon support** — The Pincer daemon now runs natively on macOS. Install via Homebrew with `brew install pincer` or use the direct install script. The release pipeline builds, signs, and publishes the macOS daemon automatically.
- **Timeline heatmap legend** — The incident timeline now includes a HeatmapLegend, making it easier to read and interpret density patterns at a glance.

## Fixes

- Improved layout consistency across agent forms, agent lists, dashboard, and settings pages with better grid and width alignment.
- Page headers and body sections now have max-width constraints for a cleaner, more consistent look.
